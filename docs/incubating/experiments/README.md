# Incubating Experiment Protocols

These protocol drafts are retained for development, not promoted as MrMaLiang
flagships. They lack one or more of: a bundled executable research runner,
end-to-end real-workload validation, a complete empirical-paper handoff, and a
detailed operator runbook comparable to the release-ready survey workflows.

Do not use them as demonstrations or production research workflows. Their
implementation configurations remain available under `packages/longexperiment`
for development and future promotion work.
